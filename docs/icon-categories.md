## UI 设计特性

### 分类筛选
- **13 个分类标签**：包含"全部"和 12 个场景分类
- **可视化标签**：每个分类带有代表性图标
- **点击切换**：点击分类标签快速筛选
- **高亮显示**：当前选中分类使用主题色高亮

### 图标展示
- **网格布局**：8列网格，紧凑高效
- **可滚动区域**：最大高度400px，支持滚动查看
- **选中状态**：选中图标带有主题色边框 (ring-2 ring-primary)
- **悬停效果**：鼠标悬停显示图标名称
- **空状态提示**：无图标时显示友好提示

### 交互体验
- **快速定位**：通过分类快速找到所需图标
- **视觉反馈**：选中、悬停状态清晰明确
- **平滑过渡**：所有状态变化都有过渡动画
